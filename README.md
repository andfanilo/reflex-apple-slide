# Challenge: Apple Keynote Slide in Python/Reflex

Reflex version, built with `rx.Box`, `rx.Flex` and [CSS Props](https://reflex.dev/docs/styling/common-props/) as Python attributes.

## Run

I prefer doing this in WSL2:

```sh
uv sync
uv run reflex run
```
